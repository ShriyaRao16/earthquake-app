# 🌍 Earthquake Monitor App

A modern **KivyMD-based Earthquake Prediction App** that allows users to enter latitude and longitude, visualize the location on a map, and get a predicted earthquake risk level.

---

## 🚀 Features

- 🗺 Interactive map with location marker  
- 📍 Reverse geolocation (shows place name)  
- 📊 Earthquake risk prediction (Low / Moderate / High)  
- 🌡 Displays estimated magnitude  
- 📏 Displays estimated depth  
- 🎨 Modern glassmorphism UI

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/ShriyaRao16/earthquake-app.git
cd earthquake-app
```
Install dependencies:
```bash
pip install -r requirements.txt
```
▶️ Run the App
```bash
python main.py
```

## 🧠 How It Works
*User enters latitude & longitude<br>
*App predicts earthquake risk using a simple logic<br>
*Location is shown on map<br>
*Address is fetched using reverse geocoding<br>
## Note
This app currently uses dummy prediction logic for demonstration purposes.
It is not intended for real-world disaster prediction.
